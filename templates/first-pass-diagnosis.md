# First-Pass Diagnosis

## Prompt

```text
Operator lens: Diagnose this creator problem before asking questions.

Problem:
[Describe the issue]

Known numbers:
[Add any numbers available]
```

## Output Template

```text
# Bottleneck
[Name the likely constraint.]

# Economic truth
[Explain what actually drives growth, money, retention, acquisition, conversion, or leverage.]

# What to stop doing
[Name the waste, distraction, false metric, or emotional trap.]

# Next measurable action
1. [Action]
2. [Action]
3. [Action]

# Scoreboard
- [Metric]
- [Metric]
- [Metric]

# Need next
- [Missing number]
- [Missing number]
```

## Example

```text
# Bottleneck
Your live stream is retaining some people, but it is not acquiring enough strangers.

# Economic truth
Engagement from existing viewers proves the product may have value. It does not prove distribution is working.

# What to stop doing
Stop using hours streamed as the main proof of growth work.

# Next measurable action
1. Pull 15 clips from the next 3 horror streams.
2. Post 10 with the scare or regret in the first 2 seconds.
3. Compare follows per 1,000 views.

# Scoreboard
- Clips shipped
- 3-second hold
- Completion rate
- Follows per 1,000 views
- Average viewers

# Need next
- Last 14 days of clips posted
- Best clip views
- Follows per stream
```
