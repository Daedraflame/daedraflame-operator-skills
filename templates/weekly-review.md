# Weekly Review

## Inputs

- Streams completed:
- Hours streamed:
- Average viewers:
- Follows:
- Clips shipped:
- Best clip:
- Discord joins:
- Active Discord members:
- Subs:
- Revenue:
- Avoided work:

## Output

```text
# Bottleneck
[One constraint.]

# What moved
[Metrics that improved.]

# What stalled
[Metrics that did not move.]

# What got avoided
[The work not done.]

# Next week's constraint
[One focus.]

# Operating plan
1. [Action]
2. [Action]
3. [Action]

# Scoreboard
[Daily/weekly metrics.]

# Need next
[Missing numbers.]
```

## Example

```text
# Bottleneck
Clip volume is too low to make a fair acquisition judgment.

# Operating plan
Stream 3 times, extract 20 clips, post 14, run one Discord vote, and review follow conversion Sunday.
```
