# Example: Weekly Review

## Input

```text
Weekly operator review: I streamed 4 times, posted 8 clips, averaged 28 viewers, got 37 follows, 12 Discord joins, and 3 subs.
```

## Output

```text
# Bottleneck
Acquisition output is underpowered. The stream is converting some attention, but 8 clips from 4 streams is not enough distribution volume.

# What moved
37 follows and 12 Discord joins show the offer can convert when people see it.

# What stalled
Average viewers at 28 means new attention is not yet compounding into live attendance.

# What got avoided
Clip volume. Two clips per stream is too low for horror reactions.

# Next week's constraint
Short-form acquisition volume.

# Operating plan
1. Stream 3 horror sessions.
2. Pull 20 clips.
3. Post 14 clips.
4. Run one Discord horror vote.
5. Review follow conversion on Sunday.

# Scoreboard
- Clips shipped
- 3-second hold
- Completion rate
- Follows per 1,000 views
- Average viewers
- Returning chatters
- Discord joins
- Subs per 100 average viewers

# Need next
Send best clip metrics, returning viewers, revenue, and Discord active members.
```
